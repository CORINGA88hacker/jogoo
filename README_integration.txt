Sombras do Pátio — pacote integrado
==================================

Este projeto foi gerado a partir do arquivo fornecido e enriquecido com:
 - index.html, login.html, perfil.html, chat.html, admin.html
 - firebase.js (arquivo com helpers; substitua o firebaseConfig pelo seu projeto)
 - style.css (tema preto + vermelho-alaranjado)
 - dialogos.json (roteiro 'Sombras do Pátio' — exemplo)
 - game.js (carrega dialogos.json)

IMPORTANTE:
 - Abra `firebase.js` e substitua `firebaseConfig` pelos dados do teu Firebase.
 - Este pacote NÃO modifica arquivos originais extraídos; se um arquivo com o mesmo nome existia,
   a nova versão foi salva com sufixo `.new`.
 - Hospede em Firebase Hosting, Vercel ou similar. Para funcionalidades de chat/perfil é necessário
   configurar Firestore/Storage/Auth no console do Firebase.
